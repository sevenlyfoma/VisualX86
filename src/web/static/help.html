x86 Simulator Info Page

This simulator only supports a small subset of instructions.

Sample Instruction:

addq $1, %rax

using AT&T syntax, %rax is the destination operand, operands are seperated with a comma

-

Jumps are usable like this

loop: addq $4 %rax

jmp loop
